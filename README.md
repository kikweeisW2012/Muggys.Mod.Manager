# Muggys.Mod.Manager
## Mod Manager and Loader for Cuphead

Muggys.Mod.Manager(MMM) Utilizes [BepinEx](https://github.com/BepInEx/BepInEx) and [MelonLoader](https://github.com/LavaGang/MelonLoader) to help manage cuphead mods
____
## Usage
1. Download Version V0.1.0
2. Extract into Cuphead root folder
3. Run Cuphead then close it
4. Run MuggysModManager.bat
5. Follow instructions
____
## Adding Basic Mods
To add BepInEx Mods add them in the folder "Cuphead\BepInEx\Plugins"
To add MelonLoader Mods add them in the folder "Cuphead\Mods"
____
## Using Auto HotKey
1. Convert your ahk file to exe
2. Place your ahk file in the Cuphead root folder
3. Rename your ahk file to "Cuphead_AHK.exe"
Now it should run while using MMM
